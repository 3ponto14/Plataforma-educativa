Política de Privacidade

3ponto14 · Plataforma educativa · Última atualização: junho de 2026

A tua privacidade é importante para nós. Esta política explica, de forma simples, que dados a 3ponto14 recolhe, para que servem e quais são os teus direitos, ao abrigo do Regulamento Geral sobre a Proteção de Dados (RGPD).

1. Que dados recolhemos

  • Progresso de estudo: pontos, exercícios feitos, respostas certas/erradas, streaks e preferências (ex.: tema claro/escuro). Por defeito, estes dados ficam guardados apenas no teu navegador (armazenamento local), no teu dispositivo.
  • Conta (opcional): se criares conta, guardamos o teu e-mail, o nome que indicares e o tipo de conta (aluno ou professor), para guardar o teu progresso e, no caso dos professores, gerir turmas.
  • Turmas (professores): quando um professor cria uma turma, ficam associados os resultados dos alunos dessa turma, para acompanhamento pedagógico.

Não recolhemos dados sensíveis nem vendemos os teus dados a terceiros.

2. Para que usamos os dados

  • Guardar e mostrar o teu progresso e as tuas conquistas;
  • Permitir que continues o estudo de onde ficaste, no mesmo dispositivo;
  • Permitir aos professores acompanhar os alunos das suas turmas;
  • Melhorar os conteúdos e o funcionamento da plataforma.

3. Onde os dados são guardados

O progresso guardado localmente fica só no teu dispositivo e pode ser apagado a qualquer momento (ver ponto 6). Os dados de conta e de turmas, quando existem, são guardados de forma segura num serviço de alojamento de dados (Supabase), com acesso restrito.

4. Crianças e menores

A plataforma destina-se ao apoio ao estudo, incluindo alunos menores de idade. O uso por menores deve ser feito com o conhecimento de um encarregado de educação ou professor. Não pedimos às crianças mais dados do que os necessários para guardar o progresso. Se um encarregado de educação quiser que os dados de um menor sejam apagados, pode contactar-nos.

5. Cookies e tecnologias semelhantes

Não usamos cookies de publicidade nem de rastreio. Usamos apenas o armazenamento local do navegador para guardar o teu progresso e preferências — é isso que permite que os teus pontos não se percam entre visitas.

6. Os teus direitos

  • Apagar o progresso local: podes limpar o teu progresso a qualquer momento na secção «O Meu Progresso» de cada curso, ou limpando os dados do site no teu navegador;
  • Aceder, corrigir ou apagar dados de conta: podes pedir-nos para ver, corrigir ou eliminar os dados associados à tua conta;
  • Retirar consentimento: podes deixar de usar a conta a qualquer momento.

7. Segurança

Adotamos medidas razoáveis para proteger os dados. No entanto, nenhum sistema é 100% seguro; recomendamos que uses uma palavra-passe forte e não a partilhes.

8. Alterações a esta política

Podemos atualizar esta política sempre que necessário. A data da última atualização está indicada no topo desta página.

9. Contacto

Para exercer os teus direitos ou esclarecer dúvidas sobre privacidade, contacta-nos através do e-mail associado à conta institucional da 3ponto14. Vê também os nossos Termos de Utilização.
